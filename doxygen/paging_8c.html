Description détaillée

Gestion de la pagination pour le système de fichiers.

Ce fichier implémente un mécanisme de pagination permettant d'allouer, de libérer et d'afficher
les pages utilisées par les fichiers dans un système de fichiers simulé.

Documentation des fonctions

◆ allocate_pages()

int allocate_pages ( int pages_needed,
PageTableEntry ** page_table )

Alloue des pages pour un fichier.

Cette fonction cherche des pages libres dans la mémoire et les assigne au fichier.
Si l'allocation échoue, toutes les pages déjà assignées sont libérées.

Paramètres
pages_neededNombre de pages à allouer.
page_tablePointeur vers la table des pages qui sera allouée.
Renvoie
0 si l'allocation réussit, -1 en cas d'échec.

◆ free_pages()

void free_pages ( File * file)

Libère les pages d'un fichier.

Cette fonction marque les pages du fichier comme libres dans la bitmap et libère sa table des pages.

Paramètres
filePointeur vers le fichier dont les pages doivent être libérées.

Affiche la bitmap de pagination.
